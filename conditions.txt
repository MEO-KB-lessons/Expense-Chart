Создать консольное приложение "Expense Chart" с возможностью построения графиков расходов по категориям с помощью matplotlib.

Пошаговая инструкция:

1. Создайте модель данных Expense с полями:

Amount
Category
Date
2. Реализуйте функционал:

Добавление, просмотр и удаление записей.
Фильтрация по категории или периоду.
Подсчёт суммы расходов за период.
3. Постройте график с использованием matplotlib для отображения расходов по категориям.

4. Используйте библиотеку datetime для обработки дат.

5. Примените принципы ООП (инкапсуляция и наследование, например, разные типы трат).

6. Реализуйте сохранение и загрузку данных в JSON.

7. Проверяйте корректность ввода:

Отрицательная сумма.
Неверный формат даты.
8. Напишите README (в формате .md), в котором укажите:

Имя и фамилию автора
Краткое описание программы
Как запустить приложение
Примеры использования (возможные тесты — по желанию)
9. Используйте Git:

Создайте локальный репозиторий.
Добавьте .gitignore.
Выложите проект на GitHub.
10. Проведите тестирование (позитивные, негативные, граничные случаи).
