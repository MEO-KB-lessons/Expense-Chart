# 💰 Expense Chart - Консольное приложение для учёта расходов

**Автор:** Иван Петров

## 📝 Описание программы

Expense Chart - это консольное приложение на Python для учёта личных расходов с возможностью визуализации данных в виде графиков. Приложение позволяет добавлять, просматривать, фильтровать и удалять записи о расходах, а также строить наглядные диаграммы расходов по категориям.

### Основные возможности:
- ✅ Добавление расходов с указанием суммы, категории и даты
- ✅ Разделение расходов по типам (обычные, важные, развлечения)
- ✅ Просмотр всех записей в табличном формате
- ✅ Фильтрация по категории, периоду или типу расхода
- ✅ Подсчёт статистики (общая сумма, средний чек, процентное соотношение)
- ✅ Построение столбчатой диаграммы расходов по категориям
- ✅ Сохранение и загрузка данных в JSON формате
- ✅ Валидация вводимых данных (сумма, дата)
